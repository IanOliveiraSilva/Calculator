## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/IanOliveiraSilva/Calculator.git
```

Entre no diretÃ³rio do projeto

```bash
  cd Calculator
```

Instale a virtualenv

```bash
  pip install virtualenv
```

Crie uma ambiente virtual

```bash
  python -m venv venv
```

Entre na pasta Scripts, ative a venv, instale o DJANGO e volte pra o diretÃ³rio raiz

```bash
  cd venv/Scripts
```
